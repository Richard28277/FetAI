## Install Instructions
1. Download zip
2. Go to command prompt, find file location
3. Open Virtual Env
```
py -m venv env
```
4. Activate the env:
```
env\Scripts\activate
```
5. Install all requirements:
```
pip install -r requirements.txt
```
7. To run: 
```
Main.py
```
